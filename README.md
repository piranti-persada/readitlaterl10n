# Pile Translations (l10n)
Hey there! 👋 Welcome to the official translations repository for Pile by Piranti Technologies!

If you speak a language other than English, you can help make Pile feel like home for people all around the world. Every contribution — big or small — means a lot to us! 🥰

All our translations are in our ARBs (Application Resource Bundles).

The English file `app_en.arb` is used as the reference. All translations are based on it! This English file uses the British spelling.

## Languages
English, Indonesian, Chinese (Hong Kong), Chinese (Simplified), Chinese (Taiwan), German, Arabic, Spanish, French, Hindi, Italian, Japanese, Korean, Dutch, Portuguese, Russian, Vietnamese.

> Don't see your language? Open an issue and let us know!

## Contribution

We'd love your help! Whether you want to fix a typo, improve a clunky phrase, or add a brand new language — you're very welcome here. 🙌
Check out CONTRIBUTING.md to get started. It's easier than you think, promise! 😊
